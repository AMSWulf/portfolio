Project:Scratch Game(Earth to Buzz)

Objective:Create a game with multiple levels and facets

Skills Learned: Importance of Collaboration and how to use Scratch more efficiently

Project:Pizza Party App
Course:CSP(Computer Science principles)
Duration:About 3 weeks(September 2018-October 2018)
Objective:Create an app from a template then modify and improve it
Type of Project: Collaborative
Skills Learned: Collaboration and how to use App Inventor

For this project I worked with my partner David to create a pizza ordering app from a template and then modify it. It started as a quick select menu app connected to a fusion table to store 'orders'. The first thing we did was add a comment box and more options. Afterwards I added prices to each item followed by a tracker of the total cost.
